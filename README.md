# .u1-reset - class
A class to reset Styles (beta)

Also intended to reset own styles:

```css
button:hover {
    transform:scale(1.2);
}

.u1-reset button:hover {
    transform:none;
}
```

**in Progress!**

## Ussage

```html
<h1>Heading 1</h1>
<div class=u1-reset>
    <h1>Heading 1</h1>
</div>
```

## Install

```html
<link href="https://cdn.jsdelivr.net/gh/u1ui/reset.class@3.0.0/reset.min.css" rel=stylesheet>
```
